# Mixery
### A Twitch inspired theme for OBS v28+ to fit the Mixer (RIPðŸ’€) integration docks

How to install the version with the custom font:

1. Download the Mixery.zip [here](https://github.com/derMiepz/Mixery/releases/tag/Release)
2. Extract Mixery.zip
3. Open Mixery (with Font) -> FONT
4. Install the font in there
5. Move the **Mixery** folder and **Yami_Mixery.ovt** file into your theme folder (Default is: ``C:\Program Files\obs-studio\data\obs-studio\themes\``)
6. Restart OBS in case it's already open
7. In OBS, go to Settings -> Appearance -> Style and select the theme
8. Click OK

Enjoy your new theme!
